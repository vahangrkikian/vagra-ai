A professional WordPress theme for law firms and legal practices with a built-in AI intake assistant.

== Description ==

Vagra Legal is a professionally designed WordPress theme built specifically for law firms and legal practices. It features a trust-focused design system with Playfair Display headings for authority and Inter body text for readability, navy and gold color scheme, and components designed to build credibility with potential clients.

Key features:

* Front page template with hero, 6 practice area cards, attorney spotlight, case results bar, testimonials, and CTA
* Built-in AI intake assistant widget powered by the Claude API (opt-in, user provides own API key)
* Legal disclaimer integration (chat preamble and footer)
* Customizer settings for brand colors, chat configuration, disclaimer text, and footer
* Responsive design with mobile-first breakpoints
* WordPress.org Theme Directory compliant
* Translation-ready with proper text domain
* Demo content import via One Click Demo Import plugin (recommended)
* Accessible with ARIA labels, roles, and semantic HTML
* 4-column footer with bar association disclaimer

== Installation ==

1. Upload the `vagra-legal` folder to `/wp-content/themes/`
2. Activate the theme through the Appearance > Themes menu in WordPress
3. Navigate to Appearance > Customize to configure theme settings
4. (Optional) Install the One Click Demo Import plugin and import demo content

== Frequently Asked Questions ==

= Does the AI chat provide legal advice? =

No. The AI chat widget is designed as an intake assistant only. It includes a mandatory disclaimer and is configured to never provide legal advice, case assessments, or specific legal recommendations. The disclaimer text is customizable in the Customizer.

= Does the AI chat require a paid API key? =

Yes, the AI chat widget uses the Claude API from Anthropic. You need to provide your own API key in the Customizer. The chat widget is opt-in and disabled by default if no API key is configured.

= Can I use this theme without the AI chat? =

Absolutely. The AI chat widget can be disabled in the Customizer. The theme works perfectly without it.
