A professional WordPress theme for MSP and Cybersecurity businesses with a built-in AI chat widget.

== Description ==

Vagra MSP Cybersecurity is a professionally designed WordPress theme built specifically for Managed Service Providers (MSPs) and cybersecurity businesses. It features a modern design system with the MLP (MSP Marketing Playbook) design tokens, a complete front page template with hero section, service cards, social proof, and call-to-action sections.

Key features:

* Front page template with hero, 6 service cards, social proof stats, and CTA section
* Built-in AI chat widget powered by the Claude API (opt-in, user provides own API key)
* Customizer settings for brand colors, typography, chat configuration, and footer
* Responsive design with mobile-first breakpoints
* WordPress.org Theme Directory compliant
* Translation-ready with proper text domain
* Demo content import via One Click Demo Import plugin (recommended)
* Accessible with ARIA labels, roles, and semantic HTML

== Installation ==

1. Upload the `vagra-msp` folder to `/wp-content/themes/`
2. Activate the theme through the Appearance > Themes menu in WordPress
3. Navigate to Appearance > Customize to configure theme settings
4. (Optional) Install the One Click Demo Import plugin and import demo content

== Frequently Asked Questions ==

= Does the AI chat require a paid API key? =

Yes, the AI chat widget uses the Claude API from Anthropic. You need to provide your own API key in the Customizer. The chat widget is opt-in and disabled by default if no API key is configured.

= Can I use this theme without the AI chat? =

Absolutely. The AI chat widget can be disabled in the Customizer. The theme works perfectly without it.

= Is this theme compatible with page builders? =

This theme is designed as a classic WordPress theme and does not require any page builder. It works with the WordPress Customizer for settings.
